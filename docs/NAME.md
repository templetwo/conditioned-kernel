# Naming

## Locked for build (2026-07-22)

| Surface | Value |
|---|---|
| Project name | **Conditioned Kernel** |
| Repo | `conditioned-kernel` |
| Python package | `conditioned_kernel` |
| CLI | `ck` / `conditioned-kernel` |
| Research synonym | Substrate-Conditioned Generation (SCG) |

## Why this name

- **Kernel** matches the architecture: the model is a replaceable transduction kernel.
- **Conditioned** names the independent variable: substrate condition, not weights.
- Short enough for a repo and CLI; ages better than a paper acronym alone.
- Leaves SCG available as the experimental method name in docs/papers.

## Alternatives considered

| Name | Notes |
|---|---|
| Substrate-Conditioned Generation | Accurate; long; kept as research synonym |
| Arrival Conditioning | Strong lineage (arrival packet); softer systems feel |
| Substrate Kernel | Short; undersells generation/experiment harness |
| Field-Conditioned Generation | Explicit CFC heir; risk of overclaim |
